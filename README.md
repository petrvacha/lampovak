# Lampovak

A conversion-focused "Fake Door" landing page for Lampovak – the first fire-resistant designer lamp filter that turns harsh light into a cozy, sleep-supporting atmosphere.

## Tech Stack
- HTML5 & CSS3 (Vanilla)
- Google Fonts (Inter)
- JavaScript (Scroll effects)

## Project Structure
- `index.html` – Core page structure
- `style.css` – Premium design and responsiveness
- `script.js` – Interactive elements
- `favicon.png` – Website icon
- `hero-bg.png` – Hero section background image

## Usage
Simply open `index.html` in any web browser.
